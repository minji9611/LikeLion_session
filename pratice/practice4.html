*form 태그 action : 데이터를 보낼 url을 지정 method : 보내는 방식을 지정 method="get" 브라우저에서 폼에 입력한 데이터를 url 끝에 붙여 눈에 보이게 보임. 주소와 데이터 함께 적어서 서버로 보냄. =받다(데이터 조회 목적 ex. 검색) method="post" 데이터가 보이지 않도록 body에 숨겨서 보냄. =게시하다(서버에 있는 데이터를 쓰거나 수정, 삭제 ex. 로그인, 게시물 작성) placeholder="" : 가이드문구 value="soonho" : 실제 할당되는 값. 데이터를 넣으면 이 속성에 값이 들어감. 초기값으로 둘 수 있음.
